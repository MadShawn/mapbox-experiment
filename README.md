# mapbox-experiment
用于mapbox-gl的学习

## 工程的环境依赖
1. vue2.0
2. vue-cli-service
3. nodejs-9.10+
4. babel-7.0+
5. yarn
6. vue-router
7. vuex
8. axios
9. normalize.css // 重置浏览器默认样式
10. element-ui // ui组件
11. veeValidate  // form验证
12. vue-easytable // 表格组件

## 工程安装
yarn install

## 工程构建
yarn build

## 工程启动
yarn serve
